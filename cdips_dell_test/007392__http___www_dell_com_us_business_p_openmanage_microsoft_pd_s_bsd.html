Enhance visibility into the health of your Dell hardware environment

Dell Server, Storage, Business Client and Printer Management Packs for Microsoft System Center Operations Manager (SCOM) deliver greater visibility into hardware inventory details, components and configurations essential to optimizing the availability of your infrastructure hardware.

Dell Server Management Pack offers both in-band and agent-free options that enable System Center Operations Manager to discover, monitor and accurately depict the status of PowerEdge servers and converged platforms, integrated Dell Remote Access Controllers (iDRACs) and Chassis Management Controllers (CMCs) on a defined network segment.

Maintain continuous availability of PowerEdge servers in virtual environments

Perform more server management tasks in less time with Dell Lifecycle Controller Integration for Microsoft System Center Virtual Machine Manager (SCVMM). Installed as a virtual appliance, this intuitive software helps you efficiently manage and maintain PowerEdge servers in virtual environments.

Dell Server PRO Management Pack integrates PowerEdge server knowledge on temperature, memory and power supplies with Microsoft System Center Virtual Machine Manager (SCVMM) and Operations Manager (SCOM) to efficiently manage Dell physical servers that host virtual workloads running on Microsoft Hyper-V. Remedial actions can be quickly implemented if a system is compromised.

Simplify and automate management tasks to boost operational efficiency

Dell Lifecycle Controller Integration for Microsoft System Center Configuration Manager (SCCM ) provides agent-free, OS- and hypervisor-agnostic configuration and deployment as well as uniform BIOS, driver and firmware updates for 11th-, 12th- and 13th-generation PowerEdge servers. Automated processes reduce the steps, time and costs needed to configure your Dell servers from a bare-metal state and to prepare remote one-to-many operating system deployments within multi-vendor operating system and hypervisor environments.

Dell Server Deployment Pack leverages OpenManage Deployment Toolkit (DTK) and PxE-based OS deployment to automate bare-metal configuration and deployment of various Microsoft operating systems on PowerEdge servers across your network.

Dell Server and Client Update Catalogs for Configuration Manager scan and update BIOS, firmware and drivers for Dell servers and business client PCs running Windows operating systems. As Dell Server and Client Update Catalogs are released, they are integrated and automatically downloaded through the System Center Configuration Manager console.
