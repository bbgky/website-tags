Help me choose: Video card

Choosing the right graphics card is critical to achieving optimal performance from graphic-intensive applications.

• Virtually every workstation application requires performance graphics.

• Dell Precision workstations offer a range of high performance graphics cards that can give you superb graphics performance for demanding applications for industries like media & entertainment, video editing, design, architecture and engineering.

• Dell Precision™ workstations offer a selection of Independent Software Vendor (ISV) certified graphics cards to support the wide range of needs for workstation users with verified solutions for high levels of reliability.

Multi-monitor support

Graphic card options provide the capability to drive 2-4 monitors by direct connection and up to 6 monitors per card using Display Port 1.2 multi-stream technology allowing compatible monitors to be connected in series with one other. Reference the Monitor Setup table for some key display support scenarios.

Dual-Graphics Options

More professional ISV applications are taking advantage of multiple GPUs in a system. Workstation users performing modeling, simulation, and rendering while working may benefit from dual-graphics configurations.
Multi-monitor use cases like display walls, security operations, and finance applications need more than 4 monitors. Dual (or more) graphics cards are used to drive and configure multiple monitors using Eyefinity software from AMD and Mosaic software from Nvidia.

- Growing business demands and shrinking development windows are challenging engineers, designers, and content creation professionals to find creative ways to explore and validate more ideas—faster. Many times this involves creating designs or content that includes a visual aspect and a physical simulation.
- Engineers, artists, designers and scientists need to be able to interact with high-performance visuals while running simulations on the same system. Workflows have changed to include multiple professional applications and graphics processors have kept pace.
- Today, professional graphics cards from AMD and Nvidia are architected to perform compute operations in parallel with more traditional graphics operations. See the compute performance table below. It is no longer necessary to have a compute + graphics card depending on your use case.
- High performance use cases can benefit from multiple graphics cards of different performance levels and cost points. Reference the table below for combinations of FirePro or Quadro / Telsa cards in Precision. When using two cards in a system it is not necessary to connect them with a dedicated cable. Data is effectively shared using the PCIe bus.
- Dell Precision workstations are designed for graphics support to meet all levels of demand.

The table below shows some examples for dual card use. Reference the power information on the recommendations tab to determine how many cards each system will support.

Precision does not offer the Nvidia Tesla K40 card. Instead use the Quadro K6000. The K6000 has slightly higher performance at a better cost point. K6000 can be used in any application instead of a K40.

In addition to using dual cards for performance (modeling, simulation, computation, and rendering for example) multiple cards are used for display walls and other multi-monitor use cases. Low cost solutions for 4 displays can use 2 graphics cards such as FirePro W2100, NVS 315 and NVS 310. For solutions of 4 or more, one or more FirePro W4100 or NVS 510 cards provide support for 8 or more displays all managed by Eyefinity (FirePro ) or Mosaic (Nvidia).
